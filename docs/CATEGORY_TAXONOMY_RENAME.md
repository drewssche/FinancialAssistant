# Category Taxonomy Rename

## Scope
Миграция нормализует названия расходных категорий без объединения `Кофе` и `Снеки`.

## Rename Map
| Kind | Старое | Новое |
|---|---|---|
| expense | Аптека/Мед услуги/БАДы | Здоровье |
| expense | Продукты/Бытовые расходники | Продукты и быт |
| expense | Кофейни/Кофе | Кофе |
| expense | Снэки/Сладости | Снеки и сладости |
| expense | Инет заказы | Онлайн-покупки |
| expense | Техника/Инструменты/Разное | Техника и инструменты |

## Data Rules
- Если новая категория уже существует, старая удаляется после переноса операций.
- Все операции с `old` автоматически переводятся на `new`.
- Удаление/переименование работает только в рамках `kind`.
